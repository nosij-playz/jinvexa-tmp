## 🧠 Core Concepts

Since we are in an AI Engineering track, we will focus on **Python**. Why? Because Python is the "lingua franca" of AI—it's readable, powerful, and has the best ecosystem of AI tools.

### 1. Variables: The "Storage Boxes"
Imagine you are moving into a new house. You have boxes labeled "Kitchen," "Bedroom," and "Books." In programming, a **variable** is just a labeled box that holds a piece of information.

*   **Integers (int):** Whole numbers (e.g., `5`, `-10`).
*   **Floats (float):** Decimals (e.g., `3.14`, `0.001`). *Crucial for AI, as most "weights" in a neural network are decimals!*
*   **Strings (str):** Text wrapped in quotes (e.g., `"Hello AI!"`).
*   **Booleans (bool):** True or False. (The "light switch" of logic).

### 2. Data Structures: The "Organizers"
Sometimes, a single box isn't enough. We need a way to group things.

*   **Lists:** Like a shopping list. An ordered sequence of items. `[ "Apple", "Banana", "Cherry" ]`
*   **Dictionaries:** Like a real dictionary. You have a **Key** (the word) and a **Value** (the definition). `{ "Name": "Alice", "Role": "AI Engineer" }`

### 3. Control Flow: The "Decision Maker"
AI is all about making decisions. We use **Conditionals** and **Loops** to handle this.

*   **If/Else Statements:** "If the image contains a cat, then label it 'Cat'. Otherwise, label it 'Not a Cat'."
*   **Loops (For/While):** "Look at every single pixel in this image one by one until you reach the end."

### 4. Functions: The "Recipe"
A function is a block of code that performs a specific task. Instead of writing the same 10 lines of code every time you want to calculate a mean average, you write a "recipe" once and call it by name whenever you need it.

## 🌍 Practical Examples: AI in Action

How do these basics actually look in an AI context?

**Scenario: A Simple Spam Filter**
Imagine we want to create a program that flags an email as "Spam" if it contains the word "Winner" and is sent from an unknown address.

```python
# 1. Variables (The data)
email_content = "Congratulations! You are the lucky winner of a million dollars!"
sender_known = False

# 2. Control Flow (The decision)
if "winner" in email_content.lower() and sender_known == False:
    status = "Spam"
else:
    status = "Inbox"

print(f"This email is marked as: {status}")
```

**Scenario: Processing a Dataset**
In AI, we often deal with lists of numbers (tensors). We use loops to process them.

```python
# A list of confidence scores from an AI model
scores = [0.92, 0.45, 0.88, 0.12]
high_confidence_predictions = []

for s in scores:
    if s > 0.5:
        high_confidence_predictions.append(s)

print(f"Reliable predictions: {high_confidence_predictions}")
```

## 📌 Key Takeaways

*   **Variables** store your data (Integers, Floats, Strings, Booleans).
*   **Lists and Dictionaries** allow you to organize complex data.
*   **Conditionals (`if/else`)** allow your program to make decisions.
*   **Loops (`for/while`)** allow you to repeat tasks efficiently.
*   **Functions** keep your code clean and reusable.
*   **Python** is the primary tool for AI because of its simplicity and vast libraries.
